# Giphyaw

This project build using Stack:

- Vite
- React
- Vitest
- Testing Library
- Tanstack React Query
- TailwindCSS
- daisyUI
- Axios
- Zod
- Giphy API

## Scripts

- How to Run ```=>``` ```yarn dev```
- How to Build ```=>``` ```yarn build```
- How to Test ```=>``` ```yarn test```
